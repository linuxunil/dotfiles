feat(config): Replace snacks.nvim explorer with oil.nvim

This commit replaces the `snacks.nvim` file explorer with `oil.nvim` to resolve startup errors and improve stability.

Key changes:
- Replaced the `snacks.nvim` explorer with `oil.nvim`.
- Removed `zen-mode.nvim`, `indent-blankline.nvim`, and `project.nvim` in favor of the equivalent `snacks.nvim` modules.
- Resolved several keybinding conflicts and redundancies.
- Cleaned up and removed several redundant and problematic autocommands.
- Updated all documentation to reflect the new changes.
